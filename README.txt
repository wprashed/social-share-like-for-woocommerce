This plugin will help you to tweet your products on twitter and share products on facebook

== Description ==

If you want to people can tweet your products on twitter, share your products on facebook and like your products using facebook. This plugin is providing all the feature. You don't need to configure anything and you don't need any API.

N.B: If you want more options in this plugin, please [contact with us](https://wprashed.com/contact "Md Rashed Hossain")

= Minimum Requirements =

* WordPress 5.0 or greater
* PHP version 7.0 or greater
* MySQL version 5.0 or greater

= We recommend your host supports: =

* PHP version 7.4 or greater
* MySQL version 5.6 or greater
* WP Memory limit of 64 MB or greater (128 MB or higher is preferred)

This section describes how to install the plugin and get it working.
